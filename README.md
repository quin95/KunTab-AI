# KunTab (WXT)

基于 WXT + React 的 Chrome 新标签页书签管理插件（V1）。

## 开发命令

```bash
npm install
npm run dev
npm run compile
npm run build
npm run zip
```

## 当前能力（V1）

- 新标签页接管（Dashboard / 书签 / 备份 / 设置）
- 书签读取、搜索、编辑、删除、移动
- 常用网站管理（添加 / 移除 / 拖拽排序）
- 最近打开记录（本地缓存）
- JSON 备份导入导出（含设置与常用网站）
- 主题、字体、启动页、搜索引擎、语言设置

详细验证见 `verification.md`。
