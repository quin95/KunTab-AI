# Bookmark AI - V1 产品需求文档（最终开发版）

------

# 一、产品概述

## 1.1 产品名称

Bookmark AI

------

## 1.2 产品定位

一个基于 Chrome 浏览器的新标签页书签管理插件。

核心目标：

- 替代 Chrome 默认新标签页
- 提供现代化书签管理体验
- 提供聚合搜索能力
- 提供常用网站快捷访问
- 完全本地运行
- 无后端
- 多设备同步（依赖 Chrome Sync）

------

# Chrome API

------

## 1）书签 API

```ts
chrome.bookmarks
```

用途：

- 获取书签
- 创建书签
- 删除书签
- 修改书签
- 获取目录树

------

## 2）Storage API

```ts
chrome.storage.sync
chrome.storage.local
```

用途：

- 保存插件配置
- 保存常用网站排序
- 保存主题设置

------

## 3）Tabs API

```ts
chrome.tabs
```

用途：

- 打开新标签
